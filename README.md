## 介绍

这是一款支持15+网赚盘解析的免费脚本，它是免费维护近两年时间的公益项目。

## 注意

我们在油猴中没有任何账号且未发布过任何脚本，谨防被骗，官方脚本只在此页面进行更新及发布。
